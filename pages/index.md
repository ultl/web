# How to create a website super easy with Nextra

## Step 1

Make sure node is installed, go to terminal

- Create a new folder for your project

```bash
mkdir web
cd web
```

- Install dependencies

```bash
npm i next react react-dom nextra nextra-theme-docs
```

- Create a `pages` folder, and an `index.mdx` inside it

```bash
mkdir pages
cd pages
code index.mdx
```

## Step 2

Grab 2 files `next.config.js` and `theme.config.tsx`, put at root folder of project.

## Step 3

At root folder of your project

```bash
npx next
```

Congratulation! Your website is up and running.
